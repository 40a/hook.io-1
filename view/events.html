Writing to Hook Events

  
To write to a Hook's logging output, simply call console.log('hello logging'); from inside the Hook.

  
Calling console.log in the Hook will automatically make the console log statement available as a logging entry.


  
Accessing Hook Events

  
To access the events for a Hook simply attach /events to the end of the Hook service url.

  
For example: https://hook.io/Marak/echo/events

  
The logging endpoint for a Hook will intelligently respond based on the incoming HTTP Accept Header.


    
Streaming events

    
To access streaming events, request the logging endpoint with an Accept Header value of */*

    
Example: curl -N https://hook.io/Marak/echo/events


    
Plain Text Events

    
To access plain text events, request the logging endpoint with an Accept Header value of text/plain

    
Example: curl -H "Accept: text/plain" https://hook.io/Marak/echo/events


  
Limitations

  
Currently all Free accounts are limited to fifty logging entries per Hook. If you need to increase this limit you can upgrade to a paid account or contact hookmaster@hook.io
