Datastore

A powerful key-value database for storing and retrieving persistent data from a Hook

Hook.io's Datastore API allows the developer to seamlessly set and retrieve documents from a persistent datastore using unique keys.

API

  • hook.datastore.set(key, value, callback)
  • hook.datastore.get(key, callback)
  • hook.datastore.del(key, callback)
  • hook.datastore.exists(key, callback)
  • hook.datastore.recent(callback)

This Datastore API is powered internally by Redis, so any native Redis commands could become availble. If you require an additional method not listed please file a support issue.

Limitations

Currently, free development accounts are limited to 1,000 unique documents per account. If you require additional documents you will need to upgrade to a paid account.

Example:

    
hook.datastore.set('mykey', 'myvalue', function(err, result){});

hook.datastore.set('/foo/anotherKey', { foo: "bar" }, function(err, result){});

hook.datastore.get('mykey', function(err, result){});

hook.datastore.incr('myCount', function(err, result){});
